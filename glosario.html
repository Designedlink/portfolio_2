TIPOS DE ETIQUETAS

En HTML y CSS tenemos diferentes tipos de etiquetas cada uno con un propósito en particular una etiqueta se puede componerse de dos maneras.

Estructura de Etiquetas
Etiquetas que solo se apertura Etiquetas que tienen una apertura y un sierre
Este tipo de etiqueta común mente se utilizan para indicar una dirección u origen se componen por argumento u opciones tales como src href. Este tipo de etiquetas común mente se utilizan para encerrar un contenido tales como texto img entre otros

Etiquetas html

  • Etiquetas dirigidas a texto

    • Titulos
    • <h1> </h1> ----------->Es un titulo de nivel uno
    • <h2> </h2> -----------> Es un titulo de nivel dos
    • <h3> </h3> -----------> Es un titulo de nivel tres
    • <h4> </h4> -----------> Es un titulo de nivel cuatro
    • <h5> </h5> -----------> Es un titulo de nivel cinco
    • <h6> </h6> -----------> Es un titulo de nivel seis
    • Parrafos
    • <p> </p> -----------> Esta etiqueta es utilizada para colocar parrafos (si se desan poner mas de un parrafo se deve de poner la misma etiqueta varias veces)
    • Listas
    • <ul> </ul> ----------->Esta etiqueta se tiliza para crear una lista desordenada
    • <ol> </ol> ----------->Esta etiqueta se tiliza para crear una lista ordenada
    • <li> </li> ------------->Esta etiqueta se tiliza para crear los elementos de una lista

  • Etiquetas dirigidas a objetos

    • <a> </a>

      • |-Esta se tiliza para colocar un vinculo o link

    • <img> </img>

      • |-Esta se utiliza para colocar imagenes o llamar imagenes

    • <img> </img>

      • |-Esta se utiliza para colocar imagenes o llamar imagenes

    • <title> </title>

      • |-Esta se utiliza para colocarle un nombre a nuestra pagina

    • <table> </table>

      • |-Esta se utiliza para colocar una tabla

    • Cajas
    • <header> </header>

      • |-Este es utilizado para definir un encabezado de página, normalmente se coloca en su interior la etiqueta <nav> </nav>.

    • <nav> </nav>

      • |-Esta etiqueta es está dirigida al menú de navegación del sitio web normalmente se encuentra dentro de la etiqueta header.

    • <head> </head>

      • |-Esta se utiliza para crear la parte principal de la pagina y provee informacion general que trae nuestra pagina

    • <section> </section>

      • |-Los sitios web están por múltiples páginas y cada página está compuesta por un header y un número indefinido de secciones en las que se tratan diferentes temas.

    • <aside> </aside>

      • |-Esta etiqueta se utiliza muy poco, se implementa cuando se desea agregar un apartado a tu sitio web como la publicidad lateral de las paginas comerciales.

    • <footer> </footer>

      • |-Esta etiqueta es utilizada para colocar un pie de página

Propiedades css

  • Propiedades css generales

    • Padding

      • Esta etiqueta es el espacio entre el contenido del elemento y su borde

    • Width

      • |-Esta etiqueta marca la anchura de las etiquetas de caja imagen

    • Height

      • |-Esta etiqueta marca la altura de las etiquetas de caja imagen

    • marning

      • Esta etiqueta establece los márgenes de un elemento especificando entre uno y cuatro valores, donde cada valor es una longitud, un porcentaje o auto.

  • Propiedades css dirigidas a texto

    • font-family

      • |-Esta etiqueta es usada para designarle un tipo de letra o tipo de fuente a los textos o títulos de nuestra página web

    • font-size

      • |-Esta etiqueta se utiliza para cambiarle el tamaño de letra o tamaño de fuente a los textos o títulos de nuestra pagina

    • text-aling

      • |-Esta etiqueta es utilizada para la posición en la que queremos que valla un texto

    • Color

      • |-Esta etiqueta es utilizada para cambiar el color de la letra

  • Propiedades css dirigidas a objetos

    • Border

      • |-Define los bordes de una etiketa

    • background-color

      • |-Esta etiqueta sirve para poder darle un color de fondo a distintas etiquetas

    • flex-wrap

      • |-Especifica si los elementos son obligados a permanecer en una misma línea o pueden fluir en varias líneas.

¿Cómo se conecta un css a un html?

La respuesta a esta pregunta es muy sencilla ya que lo que se debe de hacer es que en la etiqueta head de coloca la etiqueta link en la cual se luego se debe de agregar lo siguiente rel="stylesheet" href="css/css.css". En la etiqueta de href se coloca el nombre de la carpeta donde se encuentra tu archivo seguido del nombre que le diste a tu archivo
